Linear Regression

  • 🔍 What is Linear Regression

    Linear regression assumes that there's a straight-line relationship between the independent variables (features) and the dependent variable (target). It draws the best-fitting line through the data by minimizing the difference (errors) between actual and predicted values using Ordinary Least Squares (OLS).
    To work well, linear regression relies on
    a set of statistical assumptions—violating them means the model might give biased or inefficient estimates.

  • ✅ Assumptions of Linear Regression (OLS) – “LINE” mnemonic
    AssumptionDescriptionHow to Check
    1. LinearityRelationship between predictors and response is linearPlot residuals vs. predicted values: residuals should scatter randomly around zero
    2. IndependenceObservations are independent of each otherDurbin-Watson test (for time series); Check data collection design
    3. NormalityResiduals are normally distributed (only needed for inference, not prediction)Histogram/Q-Q Plot of residuals; Shapiro-Wilk test
    4. Equal Variance (Homoscedasticity)Residuals have constant variance at all levels of XPlot residuals vs. predicted values; Breusch-Pagan test
    5. No MulticollinearityPredictors should not be highly correlated with each otherVariance Inflation Factor (VIF): VIF > 5 or 10 is problematic
    6. No Auto-correlation (for time series)Residuals shouldn't be correlated over timeACF plots, Durbin-Watson test
  • 🧪 How to Check Each Assumption (with Code)

    Here’s a modular Python approach using statsmodels and scikit-learn:

    import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns
import statsmodels.api as sm
from statsmodels.stats.outliers_influence import variance_inflation_factor
from statsmodels.stats.stattools import durbin_watson
from statsmodels.stats.diagnostic import het_breuschpagan
from scipy.stats import shapiro, probplot

def linear_regression_diagnostics(X: pd.DataFrame, y: pd.Series):
    """Check linear regression assumptions."""
    # Add intercept
    X_const = sm.add_constant(X)
    model = sm.OLS(y, X_const).fit()
    predictions = model.predict(X_const)
    residuals = y - predictions

    print(model.summary())  # Summary statistics

    # 1. Linearity & Homoscedasticity
    plt.figure(figsize=(10, 4))
    plt.subplot(1, 2, 1)
    plt.scatter(predictions, residuals, alpha=0.5)
    plt.axhline(0, linestyle='--', color='red')
    plt.title("Residuals vs Fitted")
    plt.xlabel("Fitted values")
    plt.ylabel("Residuals")

    # 2. Normality of Residuals
    plt.subplot(1, 2, 2)
    probplot(residuals, dist="norm", plot=plt)
    plt.title("Q-Q Plot")
    plt.tight_layout()
    plt.show()

    # 3. Shapiro-Wilk test (Normality)
    stat, p = shapiro(residuals)
    print(f"Shapiro-Wilk test p-value: {p:.4f} {'(normal)' if p > 0.05 else '(not normal)'}")

    # 4. Homoscedasticity - Breusch-Pagan test
    bp_test = het_breuschpagan(residuals, X_const)
    print(f"Breusch-Pagan p-value: {bp_test[1]:.4f} {'(constant variance)' if bp_test[1] > 0.05 else '(heteroscedastic)'}")

    # 5. Multicollinearity - VIF
    vif_df = pd.DataFrame()
    vif_df["feature"] = X.columns
    vif_df["VIF"] = [variance_inflation_factor(X.values, i) for i in range(X.shape[1])]
    print("\nVIF Scores:")
    print(vif_df)

    # 6. Independence - Durbin-Watson
    dw = durbin_watson(residuals)
    print(f"Durbin-Watson statistic: {dw:.4f} (1.5 to 2.5 is acceptable)")

    🔁 Example usage:

    from sklearn.datasets import load_boston
boston = load_boston()
X = pd.DataFrame(boston.data, columns=boston.feature_names)
y = pd.Series(boston.target)

linear_regression_diagnostics(X, y)
